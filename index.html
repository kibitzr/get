Kibitzer

Self-hosted open source web monitoring tool

Tell me more!

Right for you

  • Easy to install - pip install kibitzr
  • Easy to run - kibitzr
  • Easy to configure - whole configuration in one YAML file
  • Browser scenarios with selenium
  • HTML parsing with XPath and CSS selectors
  • Integrations with MailGun and Slack
  • Highly extensible with bash and python scripts
  • Vibrant friendly community (on gitter)
  • Python 2 and 3 compatible

Easy to configure

pages:

  - name: Kibitzer Build Status
    url: https://travis-ci.org/kibitzr/kibitzr
    transform:
        - css: div.build-info > h3
        - text
        - changes
    delay: 1
    period: 10
    notify:
      - slack

  - name: Oh shit, git! update
    url: http://ohshitgit.com/
    transform:
        - tag: body
        - text
        - changes
    notify:
        - slack

  - name: New Vagrant release
    url: https://releases.hashicorp.com/vagrant/
    transform:
        - xpath: /html/body/ul/li[2]/a
        - text
        - changes
    notify:
        - mailgun

  - name: New Mr. Robot episodes
    url: http://api.tvmaze.com/search/shows?q=mrrobot
    transform:
        - json
        - changes
    notify:
        - mailgun
